## Configuration
All the credentials need to be set in environment variables.

* SNOWFLAKE_ACCOUNT_NAME
* SNOWFLAKE_USER_NAME
* SNOWFLAKE_PASSWORD
* SNOWFLAKE_ROLE
* SNOWFLAKE_DATABASE
* SNOWFLAKE_SCHEMA
* SNOWFLAKE_WAREHOUSE
